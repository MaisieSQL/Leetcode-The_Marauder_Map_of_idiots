## 🧱 第二步：创建“林克”的第一个世界 (Hello World)

我们要先让屏幕亮起来，并画出一个代表你的“蓝色小方块”。

你可以直接复制这段代码到 VS Code 中运行：

  ```python
  import pygame

  # 1. 初始化游戏引擎
  pygame.init()
  screen = pygame.display.set_mode((800, 600)) # 建立一个 800x600 像素的窗口
  pygame.display.set_caption("Wilderness of Logic - Phase 1")
  clock = pygame.time.Clock()
  
  # 2. 定义角色属性
  PLAYER_COLOR = (0, 128, 255)  # 蓝色 (RGB)
  player_pos = [400, 300]       # 初始坐标 [x轴, y轴]
  
  # 3. 游戏主循环 (这是游戏的心跳)
  running = True
  while running:
      # 检测玩家操作（比如点击关闭窗口）
      for event in pygame.event.get():
          if event.type == pygame.QUIT:
              running = False
  
      # 4. 渲染画面
      screen.fill((0, 0, 0)) # 涂背景色 (黑色)
  
      # 5. 画出主角 (一个 50x50 像素的方块)
      pygame.draw.rect(screen, PLAYER_COLOR, (player_pos[0], player_pos[1], 50, 50))
  
      pygame.display.flip() # 刷新屏幕，把画好的东西显示出来
      clock.tick(60)        # 锁定 60 帧，保证游戏运行平稳
  
  pygame.quit()
  ```
